# Stella Branded Image Generator - Quick Guide

## Access
Navigate to: **Admin → Developer → Visuals**

## Quick Steps

### 1. Select Generator Type
Click on the **⭐ Stella Branded Image** card

### 2. Review Configuration
You'll see:
- Dark blue background info
- Stella Mary logo at top
- Constellation logo + text in middle
- Ballet logo + text at bottom
- Fixed dimensions: 1200x1600px

### 3. Generate
Click the **"Generate Stella Branded Image"** button

### 4. Download
- Preview the generated image
- Click **"Download PNG"** to save
- Or click **"Copy Data URL"** for embedding

## Style Matching

The text styling **exactly matches** what's used on the Pricing page:
- **Constellation**: `text-indigo-200/80` with `0.4em` letter-spacing
- **Ballet**: `text-pink-400/90` with `0.4em` letter-spacing
- **Font**: Outfit, uppercase, sans-serif

## File Output
- **Name**: `stella-branded-{timestamp}.png`
- **Size**: 1200 × 1600 pixels
- **Format**: PNG with transparency support
- **Quality**: High resolution for print and digital
