## üéì SEO Terminology Explained

**Structured Data**: Code that helps search engines understand your content  
**Rich Snippets**: Enhanced search results with prices, ratings, images  
**Schema.org**: Standard format for structured data  
**Open Graph**: Protocol for social media link previews  
**Canonical URL**: The "official" URL for a page (prevents duplicates)  
**CTR**: Click-Through Rate - % of people who click your search result  
**SERP**: Search Engine Results Page  
**Local SEO**: Optimization for location-based searches
